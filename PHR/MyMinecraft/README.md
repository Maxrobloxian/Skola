# MyMinecraft
Je to jednoducha kopie minecraftu
## Bude tam
- Generovani terenu, biomu a mozna i dimenzii
- Hozeni, niceni, konstrukce, interactivita se svetem
- Zabava em... minecraft :D
## Engine
- Unity
- 3D
## Hlavni zamer
- Na zacatku optimizace (hra va stovky tisic voxelu(bloku) ktery tvori teren, 99% z toho neni videt)
- Pak hratelnost (pocitam s tim ze budu umirat u inventare)
## Technologie
- Unity
- C#
- pixilart.com
### 2 kapitola
- Clovek => bude mit HP, Inventar
- Itemi => treba block nebude mit nic (z pohledu playera), tooly budou mit Durabilitu, Rychlost, Attack, a jine
- teoreticky bych tu toho mel napsat vic ale uz mam 1000+ radku kody a napsat tu vsehno by moc neslo
